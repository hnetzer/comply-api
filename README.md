# Comply API
API to support Comply Hub and Comply Admin portal

### Dependencies
- Postgres 
- Node (10.x)

### Environment vars
```
PORT=8080
DATABASE=comply
DATABASE_USER=henry
```

### Database setup & seed data
```
npm install
npm run seeds
```

### Install & start
```
npm install
npm start
```
